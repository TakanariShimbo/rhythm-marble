ドラッグ範囲選択(Shiftで追加選択)

タップ修正 🥁

開始🥁ボタン。最初の対象ノートの3秒前から再生開始。打ち直す対象だけ消音(伴奏や手前のメロディは鳴る)
任意のキー叩いた瞬間に次のメロディノート(赤枠)の開始位置が打ち直される
Space / Esc途中終了(残りは元のまま)。Ctrl+Zで全部戻せる
途中から空白クリックで赤線を置いてから🥁 → それ以降だけ修正

キーボード

Space再生 / 停止
M / A選択をメロディ / 伴奏へ
L選択を左詰めで一定長に
Del選択を削除
←→↑↓選択を微調整(時間/音程)
Ctrl+A / Ctrl+Z / Ctrl+Y全選択 / 戻す / 進む
